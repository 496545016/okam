okam-cli
======

[![NPM Version](https://img.shields.io/npm/v/okam-cli.svg?style=flat)](https://npmjs.org/package/okam-cli)

> OKAM 开发框架 cli 工具


## install

```
npm i okam-cli -g
```

## Usage

```
okam init
```

```
okam upgrade self
okam upgrade project
```

```
okam help
```

## Help

```
okam help
```
